# Konversi Bilangan

Project ini adalah program sederhana untuk **konversi bilangan** yang dibuat menggunakan **Bahasa C**.  

- **main.c** : Program utama untuk menjalankan konversi bilangan.  
- **Library/Konversi.c** : Implementasi fungsi konversi bilangan.  
- **Library/Konversi.h** : Header file untuk deklarasi fungsi konversi.  
- **build/** : Folder otomatis untuk menyimpan object file dan executable.  
- **Makefile** : File untuk compile dan menjalankan program dengan mudah.

## Cara Compile dan Run
-**mingw32-make**        Untuk Compile
-**mingw32-make run**    Untuk Compile + Run
-**mingw32-make clean**  Untuk Menghapus Compile + Build

Bisa dijadikan Library tinggal Ambil dan Tuliskan Saja Konversi.c dan Konversi.h di header Project kamu!!
